# typescript-vue3-vite
  针对移动端开发，进行一体化处理。
## Item Design
1. vue3  typescript vite

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Devlopment

```sh
npm run dev
```

### Type-Check, Compile and Minfy for Production

```sh
npm run build
```

### Lint width [ESLint](https://eslint.org/)

```sh
npm run lint
```
